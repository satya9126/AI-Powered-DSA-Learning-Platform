## Overview
AlgoMentor is a web-based platform designed to help students learn Data Structures and Algorithms effectively.

It provides structured coding problems, a built-in code editor, and AI-based guidance such as hints, error detection, and step-by-step explanations. The system supports self-paced learning and improves problem-solving skills.

---

## Key Features
- Topic-wise DSA problems  
- Difficulty levels (Easy, Medium, Hard)  
- Built-in code editor  
- Code execution and validation  
- AI-based hints and error detection  
- Step-by-step solutions  
- Progress tracking
